## 一、审核红线与一票否决项（永久生效）
1.  合规红线：文章存在敏感内容、违规内容、侵权风险，审核结果直接为不通过
2.  格式红线：标题超出26字、配图少于3张、存在html乱码、图片堆叠，审核结果直接为不通过
3.  内容红线：文章完全脱离选定主题、存在编造虚假信息、夸大技术效果，审核结果直接为不通过
4.  **标准缺失红线**：审核前未通读 tech-html-acceptance-standards-1.md 的阶段4，审核结果直接为不通过。审核必须逐项对标10项标准清单执行，不允许打综合分，不允许模糊通过。

## 六、审核标准参考（阶段4核心条款，2026-04-03确认精简为仅读此节）

**10项审核清单：**
1. 错别字/病句 → 0错别字、0病句、标点规范
2. 技术数据准确性 → 模型版本/价格/API规则与权威来源一致
3. 合规安全 → 0敏感词、0违规内容、0侵权风险
4. 配图数量与分布 → 至少3张，均匀分布在开头/中间/结尾
5. 配图链接有效性 → 全部为mmbiz.qpic.cn永久链接
6. 代码块格式 → Plain Text格式，非blockquote
7. HTML干净度 → 无乱码、无残留标签
8. 链接有效性 → 内链/参考资料可正常打开
9. 字数 → 正文900-1100字
10. 标题 → 14-26字
