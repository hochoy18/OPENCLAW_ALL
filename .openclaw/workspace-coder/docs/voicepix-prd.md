## 2. 背景与目标

### 2.1 背景

VoicePix 是一款基于 MiniMax Speech API 的网页端文字转语音（T2A）及声音克隆工具。当前已完成 MVP 阶段，产品以单文件 HTML + Vanilla JS 的轻量化架构交付，部署于 Cloudflare Tunnel。

**当前存在的问题与机会：**

- 用户缺乏便捷的网页端 T2A 工具，需依赖桌面客户端或复杂 SDK
- 市场上现有 T2A 网页工具音色选择有限，中英双语支持不足
- 声音克隆功能需要一定技术门槛，用户无法快速生成自定义音色
- API Key 管理不便，用户每次使用需重复配置

### 2.2 目标

**业务目标：**
- 为用户提供一款**免费、免安装、浏览器即用**的网页端 T2A 及声音克隆工具
- 支持**19种预设音色**和**中英双语**，覆盖主流配音场景
- 提供**自定义声音克隆**能力，降低 AI 音色定制的门槛

**用户价值：**
- 内容创作者快速生成配音旁白
- 开发者测试和集成 MiniMax T2A API
- 个人用户探索声音克隆的趣味玩法

### 2.3 成功指标

| 指标 | 描述 | 目标值 |
|------|------|--------|
| 功能可用性 | T2A 生成成功率 | ≥ 95% |
| 功能可用性 | 声音克隆成功率 | ≥ 90% |
| 用户体验 | 页面首次加载时间 | ≤ 3s |
| 用户体验 | API 响应反馈时间 | ≤ 5s（含网络延迟）|
| 部署稳定性 | 服务可用性 | ≥ 99%（Cloudflare Tunnel）|

## 3. 用户场景

### 3.1 目标用户

| 用户画像 | 描述 |
|---------|------|
| 内容创作者 | B站/抖音/小红书 up 主，需要为视频生成配音旁白 |
| 自媒体运营 | 需要快速生成多风格配音，测试不同音色效果 |
| 开发者 | 调试和测试 MiniMax T2A API，集成到自有应用 |
| 尝鲜用户 | 对声音克隆感兴趣，希望用少量音频样本定制个人音色 |

### 3.2 用户故事

- **作为内容创作者**，我希望选择不同音色生成配音，以便快速产出多版本文案配音样稿。
- **作为开发者**，我希望直接在前端调试 T2A API 参数，以便快速验证音色和参数效果。
- **作为尝鲜用户**，我希望上传自己的音频样本克隆声音，以便用自己或朋友的声音生成语音。
- **作为运营者**，我希望调整语速和音调，以便生成不同情绪氛围的配音效果。

### 3.3 使用场景

**场景1：快速生成配音**
- 触发条件：用户输入文字，选择音色，点击生成
- 用户行为：粘贴文案 → 选择音色 → 调整语速/音调 → 点击"生成" → 预览/下载音频
- 预期结果：生成对应音频文件，支持 mp3/wav/ogg/flac 四种格式下载

**场景2：声音克隆**
- 触发条件：用户上传音频样本，填写音色名称，提交克隆请求
- 用户行为：上传音频（≥30s） → 填写音色名称 → 点击"克隆" → 等待生成 voice_id → 使用新音色生成 T2A
- 预期结果：生成自定义 voice_id，出现在音色列表中，可用于 T2A 生成

**场景3：API Key 配置**
- 触发条件：用户首次使用或 API Key 过期/变更
- 用户行为：打开设置 → 输入 API Key → 保存 → 本地存储至浏览器
- 预期结果：API Key 持久化存储，后续使用无需重复输入

### 4.2 功能详情

#### 4.2.1 文字转语音（T2A）

**需求描述：**
用户输入文本，选择预设音色或克隆音色，调节语速/音调，选择输出格式，发起 T2A 请求，获取生成的音频。

**交互流程：**
1. 用户在文本框输入或粘贴文案（支持中英文混合）
2. 用户从音色列表中选择目标音色（预设19种 + 自定义克隆音色）
3. 用户调节语速滑块（慢/正常/快）和音调滑块（低/正常/高）
4. 用户选择输出格式（mp3 / wav / ogg / flac）
5. 用户点击"生成"按钮
6. 系统调用 MiniMax T2A API，显示加载状态
7. API 返回音频后，页面展示音频播放器，支持在线试听和下载

**页面元素：**

| 元素 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 文本输入框 | textarea | 输入配音文案 | 必填，最大 2000 字符 |
| 音色选择器 | 下拉列表 | 19种预设 + 克隆音色 | 必填 |
| 语速调节 | 滑块/选择器 | 慢/正常/快 | 默认正常 |
| 音调调节 | 滑块/选择器 | 低/正常/高 | 默认正常 |
| 输出格式 | 单选按钮组 | mp3/wav/ogg/flac | 默认 mp3 |
| 生成按钮 | button | 提交 T2A 请求 | 文本为空时禁用 |
| 音频播放器 | audio | 展示生成结果 | 生成成功前隐藏 |

**异常处理：**
- 网络超时：提示"网络超时，请重试"，保留用户输入
- API Key 无效：提示"API Key 无效，请检查配置"，引导跳转设置
- 余额不足：提示"账户余额不足，请充值"
- 文本超长：前端截断并提示"文本过长，已截断至2000字符"
- 生成失败：提示具体错误信息，保留用户输入和参数选择

**埋点需求：**
- 事件 `t2a_generate`：触发时机 - 点击生成按钮，携带参数（音色ID、语速、音调、格式、文本长度）
- 事件 `t2a_success`：触发时机 - 生成成功，携带参数（耗时、音色ID）
- 事件 `t2a_error`：触发时机 - 生成失败，携带参数（错误类型）

#### 4.2.2 声音克隆

**需求描述：**
用户上传音频样本（≥30秒），填写音色名称，发起克隆请求，系统调用 MiniMax Voice Clone API，生成自定义 voice_id，并将其添加到音色列表中供后续使用。

**交互流程：**
1. 用户点击"声音克隆" Tab 或区域
2. 用户上传本地音频文件（支持 wav/mp3，建议 ≥30秒）
3. 用户填写自定义音色名称
4. 用户点击"开始克隆"按钮
5. 系统上传音频文件，调用 Voice Clone API
6. 页面展示上传进度 + 克隆处理进度
7. 克隆成功后，自定义音色出现在音色下拉列表中
8. 用户可立即使用新音色进行 T2A 生成

**页面元素：**

| 元素 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| 音频上传区 | file input | 支持拖拽或点击上传 | 必填，支持 wav/mp3，≤ 10MB |
| 音色名称输入 | text input | 自定义音色名称 | 必填，2-20字符，不重复 |
| 克隆按钮 | button | 提交克隆请求 | 音频或名称为空时禁用 |
| 进度条 | progress | 展示上传和克隆进度 | 克隆完成后消失 |
| 克隆音色列表 | 下拉列表 | 展示已克隆的自定义音色 | 可删除 |

**异常处理：**
- 音频时长不足：提示"音频时长不足，请上传至少30秒的音频"
- 音频格式不支持：提示"仅支持 wav/mp3 格式"
- 音频过大：提示"音频文件过大，请控制在 10MB 以内"
- 克隆失败：提示具体错误信息，清空当前上传和名称输入
- 名称重复：提示"该音色名称已存在，请使用其他名称"

**埋点需求：**
- 事件 `clone_upload`：触发时机 - 上传音频文件，携带参数（文件大小、时长）
- 事件 `clone_start`：触发时机 - 点击克隆按钮
- 事件 `clone_success`：触发时机 - 克隆成功，携带参数（耗时、voice_id）
- 事件 `clone_error`：触发时机 - 克隆失败，携带参数（错误类型）

#### 4.2.3 API Key 配置

**需求描述：**
用户首次使用或更换 API Key 时，在设置页面输入并保存 MiniMax API Key。API Key 存储于浏览器 localStorage，页面刷新后自动填充。

**交互流程：**
1. 用户点击"设置"入口
2. 用户在 API Key 输入框粘贴自己的 MiniMax API Key
3. 用户点击保存按钮
4. 系统校验 Key 格式（非空，长度符合标准格式）
5. 保存至 localStorage，关闭设置面板
6. 后续访问自动从 localStorage 读取并填充

**页面元素：**

| 元素 | 类型 | 说明 | 校验规则 |
|------|------|------|---------|
| API Key 输入框 | password input | 隐藏式输入 | 必填，格式校验 |
| 显示/隐藏切换 | toggle | 显示明文或掩码 | 默认隐藏 |
| 保存按钮 | button | 保存配置 | Key 为空时禁用 |
| 测试连接按钮 | button | 验证 Key 是否有效 | 可选 |

**异常处理：**
- Key 格式错误：提示"API Key 格式不正确，请检查"
- localStorage 满：提示"存储空间不足，请清理浏览器缓存"

## 6. 接口说明

### 6.1 MiniMax T2A API

**接口信息：**
- Endpoint：`https://api.minimax.io/v1/t2a_v2`
- Method：POST
- 文档：MiniMax 官方 T2A API 文档

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| model | string | 是 | 模型标识 | `speech-02-hd` |
| text | string | 是 | 待转换文本 | `你好，欢迎使用 VoicePix` |
| stream | boolean | 否 | 是否流式返回 | `false` |
| voice_setting | object | 是 | 音色设置 | 见下表 |
| audio_setting | object | 是 | 音频设置 | 见下表 |

**voice_setting 子参数：**

| 参数名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| voice_id | string | 音色 ID | `male-qn-qingse` |
| speed | float | 语速，0.5-2.0 | `1.0` |
| pitch | float | 音调，0.5-2.0 | `1.0` |

**audio_setting 子参数：**

| 参数名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| sample_rate | int | 采样率 | `32000` |
| bitrate | int | 比特率 | `128000` |
| format | string | 输出格式 | `mp3` |

**响应格式：**
- Content-Type: `audio/mpeg`（或其他指定格式）
- 返回二进制音频数据

**错误响应：**

| HTTP 状态码 | 错误信息 | 说明 |
|-------------|---------|------|
| 401 | Unauthorized | API Key 无效 |
| 429 | Rate limit exceeded | 请求频率超限 |
| 400 | Bad request | 参数错误或余额不足 |

---

### 6.2 MiniMax Voice Clone API

**接口信息：**
- Endpoint：`https://api.minimax.io/v1/voice_clone`
- Method：POST
- Content-Type：`multipart/form-data`
- 文档：MiniMax 官方 Voice Clone API 文档

**请求参数：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| voice_name | string | 是 | 自定义音色名称 |
| audio_file | file | 是 | 音频样本文件（wav/mp3） |

**响应格式：**

```json
{
  "voice_id": "custom_voice_xxxx",
  "voice_name": "我的声音",
  "status": "success"
}
```

**错误响应：**

| HTTP 状态码 | 错误信息 | 说明 |
|-------------|---------|------|
| 400 | Bad request | 参数错误（音频格式/大小/时长不符）|
| 401 | Unauthorized | API Key 无效 |
| 429 | Rate limit exceeded | 请求频率超限 |

## 7. 风险评估

| 风险类型 | 风险描述 | 影响程度 | 应对措施 |
|---------|---------|---------|---------|
| **技术风险** | MiniMax API 域名或接口路径变更 | 中 | 预留 API 配置化开关，支持快速切换 |
| **技术风险** | 单文件架构扩展性受限 | 低 | MVP 阶段可接受，后续可重构为多模块 |
| **业务风险** | 用户 API Key 安全问题（泄露）| 高 | 前端不传输 Key 至第三方，仅本地存储 |
| **业务风险** | MiniMax API 收费涨价或调整配额 | 中 | 明确告知用户费用由用户自行承担 |
| **合规风险** | 用户克隆他人声音的伦理/法律风险 | 高 | 在界面添加免责声明：仅供研究学习，勿用于违法场景 |
| **运维风险** | Cloudflare Tunnel 服务中断 | 中 | 监控 Tunnel 状态，准备备用部署方案 |
| **运维风险** | 浏览器 localStorage 被清除 | 低 | 引导用户定期备份 Key，或提供导出功能 |
