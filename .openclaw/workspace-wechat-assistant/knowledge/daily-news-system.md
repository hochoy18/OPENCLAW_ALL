## 一、项目背景

### 1.1 目标
实现**每天北京时间 7:00** 自动发现热点、创作公众号文章、生成封面图、推送草稿箱的全流程自动化。

### 1.2 内容方向
| 方向 | 占比 | 说明 |
|------|------|------|
| AI 工具测评 | 70% | 实测 + 独立观点 |
| 职场打工人 | 30% | 积极向上风格 |

### 1.3 创作原则
- **实测为主**：AI 测评必须自己调用 API 实际操作
- **原创优先**：不搬运、不抄袭
- **积极正向**：职场内容不吐槽、不负能量
- **安全边际**：不碰政策敏感话题

### 4.2 详细步骤说明

#### Step 1-2: 热点发现与选题

**搜索关键词（AI 方向）**：
```
AI tools trending this week
new AI tool release
ChatGPT alternatives
AI productivity tools
best AI apps 2024
人工智能工具 最新
AI 软件 热点
```

**搜索关键词（职场方向）**：
```
职场 热议 2024
打工人 热点
职场 成长 干货
工作 技巧
```

**选题决策规则**：
1. 优先选实测可行的题目
2. 选有独特视角的题目（不是泛泛而谈）
3. 排除：太旧的话题、太窄的细分、争议性太强的

#### Step 3-4: 深度搜索与实测

**实测执行**：
- 如果是 AI 工具，真的调用 API 使用
- 记录：速度、效果、优点、缺点
- 给出明确评价

**无法实测时**：
- 改为介绍类内容
- 侧重于：是什么、怎么用、适合谁

#### Step 5: crayon 写作规范

**必须遵守**：
- 开头 3 段内有"冲突/反差/损失感"钩子
- 每 4-6 段插入小标题
- 至少给 1 组 if/then 判断
- 结尾必须有 CTA

**禁用表达**：
- "赋能、抓手、闭环、矩阵化"
- "综上所述、毋庸置疑、不难发现"
- 连续三段以上同构句式

#### Step 6: 封面图生成

**Prompt 模板**：
```
文章封面：[标题]。风格：简洁现代、高质量插画、16:9横屏构图、强对比、高可读中文标题排版。主视觉：[核心意象]。颜色：[主色+强调色]。避免：复杂背景、过多文字、水印。
```

**执行命令**：
```bash
python3 generate_cover.py \
  --prompt "文章封面：..." \
  --provider minimax \
  --aspect-ratio "16:9" \
  --output /tmp/cover.jpg
```

#### Step 6.5: 正文配图生成 ▲ 新增

**执行时机**：文章写完后、发布前

**配图规则**：
- 数量：2-3 张（根据文章长度）
- 位置：主要段落结束后插入
- 尺寸：横屏 16:9（900×383 或 1280×720）
- 风格：与文章内容相关、高质量插画

**配图 Prompt 模板**：
```
文章配图：[段落主题描述]。风格：简洁现代插画、16:9横屏构图、高质量、与文章内容高度相关。避免：文字过多、复杂背景。
```

## 九、FAQ

**Q: 为什么不直接发布？**
A: 微信限制，正式发布不可逆。需要用户审稿确认。

**Q: 正文配图是怎么实现的？**
A: 
1. 文章写完后，分析段落结构确定配图位置
2. 根据每段内容生成对应 Prompt
3. MiniMax image-01 生成图片
4. 上传到微信 CDN（mmbiz.qpic.cn 域名）
5. 插入 HTML 对应位置

**Q: 配图数量多少合适？**
A: 2-3 张，根据文章长度调整。太多会影响加载速度。

**Q: 如果配图不匹配怎么办？**
A: 用户在审稿时可以替换或删除，我生成的只是初稿。

**Q: 如果热点搜不到怎么办？**
A: 使用备选关键词重试，或改为通用话题。

**Q: 怎么暂停/修改？**
A: 使用 `openclaw cron edit fd5f86da-6fac-4820-885e-e11382dd0d67` 修改。
