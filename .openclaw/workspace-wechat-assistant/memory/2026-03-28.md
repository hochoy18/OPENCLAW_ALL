## 💡 经验沉淀

- **公众号草稿 API**: 必须用 `material/add_material`（永久素材），不能用 `media/upload`（临时素材）
- **Cron 超时**: 内容创作类任务（写作+生图+上传）需要更长 timeout，建议 ≥1800 秒
- **Isolated session + Feishu 消息**: 必须指定 `accountId="main"`，因为 isolated session 无法自动路由到正确的飞书账号
- **announce 模式**: 对 isolated session 无效，需要改用 `mode: "none"` + 手动发送消息
