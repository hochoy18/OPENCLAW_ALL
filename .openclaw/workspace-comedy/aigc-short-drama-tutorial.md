# AIGC微短剧制作教程

## 一、制作流程

### 1. 选题策划
- 热门题材：甜宠、逆袭、悬疑、古装、职场
- 公式：强冲突 + 高共鸣 + 快节奏 + 悬念钩子

### 2. 剧本创作（3分钟结构）
```
0:00-0:10  钩子：冲突爆发
0:10-0:40  铺垫：背景介绍
0:40-1:30  发展：矛盾升级
1:30-2:20  高潮：最大冲突
2:20-2:50  结局：留悬念
2:50-3:00  CTA：引导关注
```

### 3. 角色设计
- Midjourney生成角色定装照
- 使用--seed或--cref保持角色一致性

### 4. 分镜脚本
| 镜号 | 时间 | 景别 | 画面 | 配音 |
|------|------|------|------|------|
| 1 | 0-5s | 特写 | 女主震惊 | 惊呼声 |

### 5. 画面生成
- 工具：Midjourney/SD/即梦/可灵
- 流程：角色定装照 → 场景图 → 分镜画面

### 6. 配音制作
- 工具：MiniMax/ElevenLabs/Edge TTS
- 用Whisper生成时间戳字幕

### 7. 音乐音效
- 音乐：Suno/Udio/剪映
- 音效：freesound/爱给网

### 8. 视频合成（剪映）
1. 导入素材
2. 排列图片
3. 添加转场
4. 添加字幕
5. 调整时长匹配配音
6. 添加音乐音效
7. 调色
8. 导出（1080×1920, 30fps）

## 二、工具推荐

| 环节 | 工具 |
|------|------|
| 剧本 | ChatGPT/Claude/Kimi |
| 角色 | Midjourney/SD/即梦 |
| 画面 | MJ/SD/可灵/Runway |
| 配音 | MiniMax/ElevenLabs |
| 音乐 | Suno/Udio/剪映 |
| 合成 | 剪映/PR/Remotion |

## 三、关键技巧

1. **角色一致性**：固定seed或用--cref
2. **节奏把控**：每30秒一个反转
3. **开头3秒**：必须有冲突或悬念
4. **字幕同步**：用Whisper生成真实时间戳
5. **音量平衡**：配音 > 音乐
