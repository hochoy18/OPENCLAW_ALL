### 视频制作项目（OpenClaw 宣传视频）
- **用户需求**：制作一个 OpenClaw 介绍短视频（竖屏 9:16，带英文 TTS 旁白）
- **技术方案**：
  1. 通过 Tavily 搜索 OpenClaw 官方文档获取内容
  2. 用 Python PIL 生成4张渐变色风格幻灯片（1080x1920）
  3. 用 minimax-speech TTS 生成英文旁白（11.3秒）
  4. 用 ffmpeg concat 将幻灯片 + 音频合并为 WebM 视频
- **产出**：`video_project/openclaw_final.webm`（255KB，11.3秒）
- **后续**：用户要求转 MP4，服务器无 H.264 软件编码器，用 mpeg1video 制作了 `openclaw_mpeg1.mp4`（1.9MB）
