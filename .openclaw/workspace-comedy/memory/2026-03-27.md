2. **用户 Plus $20/月套餐解析**
   - Speech 2.8：4000字符/天
   - image-01：50张/天
   - M2.7：1500请求/5小时
   - 无视频生成、无音乐生成

3. **两个场景的具体分析**
   - 小程序音效：完全能做，4000字符/天额度绑绑有余
   - 短视频旁白配音：基本能做，每天2~5条
   - 短视频背景音乐：不能做（需升级Max套餐）

4. **关键限制提醒**
   - Token Plan API Key ≠ 普通按量计费 Key
   - 额度重置周期不同（文本5小时滚动，语音/图片每日零点）
   - 音色复刻7天有效期，需定期续命
   - 10个快速克隆音色名额

## 决策与结论
- 推荐用户先用 Plus 套餐跑小程序音效场景，短视频旁白量不大也够用
- 高频短视频旁白建议升级 Max 套餐
- 待用户确认后可直接生成音效样本
