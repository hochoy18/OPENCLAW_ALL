## Day 17：性能优化

### Day 17-1：包体优化（目标：主包≤4MB）

**Claude Code 指令**：
```
在 Cocos Creator 中优化包体：

1. 菜单栏 → 项目 → 项目设置 → 功能裁剪
   - 关闭不需要的模块：
     * 3D（本案为2D游戏）
     * 视频播放（如未使用）
     * WebView（如未使用）
     * 尽量精简

2. 资源配置：
   - 图片使用压缩格式（ASTC）
   - 合图（Sprite Atlas）：将小图标合并为大图
   - 关闭调试图片资源

3. 代码优化：
   - 确认无无用代码引用
   - 第三方库按需加载

4. 构建测试：
   - 菜单栏 → 项目 → 构建发布
   - 平台：微信小游戏
   - 检查构建输出大小
   - 如果主包超过 4MB，使用分包（第2/3章做分包）

完成后报告主包大小。
```

### Day 17-2：运行性能优化

**Claude Code 指令**：
```
在 PhysicsManager.ts 中优化性能：

1. 开启物体休眠：
   - 所有静态/休眠刚体在碰撞后自动进入休眠
   - 减少物理计算量

2. 限制每关物理体数量：
   - 在 LevelManager 中设置 MAX_OBJECTS = 50
   - 超出自禁止继续放置

3. 粒子特效优化：
   - 限制同屏粒子数量 ≤ 100
   - 特效播放后自动销毁节点

4. 场景切换优化：
   - 关卡切换时，先清空上一关所有节点，再加载新关卡
   - 防止节点残留

完成后报告性能优化情况。
```
