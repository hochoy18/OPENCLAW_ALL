## 一、游戏概述

### 1.1 一句话定位

一款"建造+投掷+破坏"融合的物理休闲游戏。玩家在有限资源下搭建结构，用投掷物精准打击目标，体验"建中有拆、拆中有建"的策略操作快感。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 性别 | 男性为主（约85%） |
| 年龄 | 18-35岁 |
| 画像 | 有一定游戏经验，不满足纯无脑点击，追求"操作有回报、策略有价值" |
| 心理需求 | 在"破坏"中发泄，在"建造"中掌控，在"成长"中满足 |

### 1.3 核心Hook

| Hook | 设计 |
|------|------|
| 操作爽感 | 每一投都有真实物理反馈，命中目标时有"WOW"破坏特效 |
| 策略深度 | "用什么投、投哪里，先建还是先拆"不是无脑选择 |
| 成长曲线 | 永久升级系统，每局都有进度感 |
| 每日挑战 | 每天1个特殊关卡，全服排名，重复动力 |
| 社交传播 | 通关生成分数截图，微信一键分享 |

### 1.4 主题融合

| 元素 | 具体内容 |
|------|---------|
| 投掷物 | 拖鞋（日常幽默）、哑铃（工业力量）、杠铃片、烟花弹 |
| 场景 | 工地、阳台、健身房、办公室、仓库 |
| 建造材料 | 木板、钢梁、绳索 |
| 视觉基调 | 暖黄（#F5A623）+ 深灰（#4A4A4A）+ 橙色（#FF6B35） |

## 二、核心玩法循环

### 2.1 宏观游戏循环

```
启动游戏
    ↓
每日挑战（重复动力）
    ↓
进入章节关卡
    ↓
【建造阶段】（约40%游戏时间）
  - 选择材料，在预算内搭建结构
    ↓
【投掷阶段】（约50%游戏时间）
  - 选择投掷物 → 瞄准 → 蓄力 → 发射
  - 物理碰撞 → 结构倒塌 → 目标破坏
    ↓
【结算阶段】（约10%游戏时间）
  - 星级评价（1-3星）
  - 资源奖励
  - 升级解锁
    ↓
返回 → 下一关 or 每日挑战
```

### 2.2 建造阶段

| 设计要点 | 描述 |
|---------|------|
| 材料限制 | 每关有"材料预算"上限，不能无脑堆料 |
| 材料种类 | 木板（轻）、钢梁（重强）、绳索（受拉）、炸药（爆破） |
| 操作 | 拖拽放置 + 旋转角度 + 确认 |
| 撤销 | 每关3次撤销（后悔药机制） |

### 2.3 投掷阶段

| 设计要点 | 描述 |
|---------|------|
| 瞄准辅助 | 显示投掷轨迹预测线（可开关） |
| 蓄力机制 | 按住越久力越大，有上限 |
| 投掷物 | 石块/哑铃/杠铃片/烟花弹（各不同物理属性） |
| 连击 | 一次投掷触发多次碰撞 = 额外分数 |

### 2.4 破坏结算

- 投掷物碰撞结构 → 物理模拟移动/断裂
- 支撑点被破坏 → 连锁倒塌
- 全部目标摧毁 → 完美过关 → 3星

## 三、游戏内容结构

### 3.1 章节体系（3章30关）

| 章节 | 名称 | 核心教学 | 场景 | 关卡 |
|------|------|---------|------|------|
| 第1章 | 物理启蒙 | 投掷基础、抛物线 | 工地/沙坑 | 10关 |
| 第2章 | 结构秘密 | 搭桥原理、支撑点 | 乡村小路/小河 | 10关 |
| 第3章 | 破坏艺术 | 精准打击、连锁破坏 | 废弃工厂/仓库 | 10关 |

每章结构：第1-7关教学 → 第8-9关复习组合 → 第10关章节Boss

### 3.2 难度参数曲线

| 关卡范围 | 设计参数 | 难度 |
|---------|---------|------|
| 第1-5关 | 预算充足 + 1个目标 + 仅石块 + 简单堆叠 | ★★☆☆☆ |
| 第6-10关 | 预算中等 + 2个目标 + 引入建造 | ★★★☆☆ |
| 第11-15关 | 预算紧张 + 2个目标 + 需规划支撑点 | ★★★★☆ |
| 第16-20关 | 预算极紧 + 3个目标 + 精准打击 | ★★★★★ |
| 第21-30关 | 综合技能全用 + 每章第10关为章节Boss | ★★★★★ |

### 3.3 投掷物系统

| 投掷物 | 物理特点 | 解锁条件 | 获取方式 |
|--------|---------|---------|---------|
| 石块 | 基础、便宜、中等伤害 | 初始自带 | 初始解锁 |
| 哑铃 | 重、伤害高、弹跳少 | 第5关解锁 | 关卡解锁 |
| 杠铃片 | 极重、可压毁硬结构 | 第10关解锁 | 关卡解锁 |
| 烟花弹 | 范围爆炸+点燃效果 | 第15关解锁 | 关卡解锁 |

### 3.4 建造材料系统

| 材料 | 物理特点 | 预算花费 |
|------|---------|---------|
| 木板 | 轻、中等支撑力 | 10金币/块 |
| 钢梁 | 重、高支撑力 | 30金币/根 |
| 绳索 | 受拉、不能受压 | 20金币/根 |
| 炸药 | 破坏地形、创造通道 | 50金币/包 |

### 3.5 升级系统（永久成长）

| 升级路线 | 效果 | 上限 |
|---------|------|------|
| 投掷力+ | 每级+5% | 20级 |
| 精准度+ | 轨迹预测更精确 | 20级 |
| 资源获取+ | 每关金币+10% | 20级 |
| 撤销次数+ | 每关+1次撤销 | 3次 |

### 3.6 每日挑战

| 要素 | 设计 |
|------|------|
| 内容 | 每天1个特殊高难度关卡 |
| 排行 | 全服按分数排名（本地存储，无后端） |
| 奖励 | 金币 + 特殊头像框 |
| 分享 | 一键生成炫耀截图发微信 |

### 3.7 社交系统

| 功能 | 设计 |
|------|------|
| 微信分享 | 过关后生成分数截图，带游戏水印 |
| 好友排行 | 微信关系链，本地存储比较（无后端服务） |
| 邀请奖励 | 邀请好友，双方获得50金币（v1暂不实现，广告变现后补充） |

const PHYSICS_CONFIG = {
  gravity: { x: 0, y: -2000 },  // 适合抛物线体验

### 7.2 投掷系统（轨迹预测）

```typescript
function predictTrajectory(start: Vec2, velocity: Vec2, steps: number): Vec2[] {
  const result: Vec2[] = [];
  const dt = 1/60;
  let pos = start.clone(), vel = velocity.clone();
  for (let i = 0; i < steps; i++) {
    vel.y += PHYSICS_CONFIG.gravity.y * dt;
    pos.x += vel.x * dt;
    pos.y += vel.y * dt;
    result.push(pos.clone());
  }
  return result;  // 绘制虚线
}
```

### 7.3 破坏检测

```typescript
onContactBegin(self: Collider2D, other: Collider2D) {
  const force = self.rigidBody.linearVelocity
    .sub(other.rigidBody.linearVelocity).length()
    * self.rigidBody.getMass();
  if (force > other.targetDestructionThreshold) {
    other.node.destroy();
    ParticleFX.play(other.node.position, 'debris');
    this.evaluateLevelComplete();
  }
}
```

### 7.4 关卡参数化（速度优先的核心）

```typescript
function generateLevelVariant(templateId: number, difficulty: number): LevelData {
  const template = LEVEL_TEMPLATES[templateId];
  return {
    ...template,
    budget: template.budget * (1 - difficulty * 0.1),
    targetCount: template.targetCount + Math.floor(difficulty / 3),
    availableProjectiles: template.availableProjectiles.slice(
      0, Math.min(4, 1 + Math.floor(difficulty / 5))
    ),
  };
}
```
