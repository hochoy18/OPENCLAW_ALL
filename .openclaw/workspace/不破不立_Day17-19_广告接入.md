**Claude Code 指令**：

## Day 18：5个激励视频广告场景

### Day 18-1：广告管理器

**Claude Code 指令**：
```
创建广告管理系统：

创建文件 assets/scripts/ads/AdManager.ts：

1. 激励视频（RewardedVideo）：
   - 当前加载的广告实例
   - preLoad(): 预加载下一个激励视频
   - show(adUnitId: string, onClose: (completed: boolean) => void)
     播放激励视频，完成回调
   - 使用穿山甲 SDK API

2. 插屏广告（InterstitialAd）：
   - 每3关计数器
   - shouldShowInterstitial(): boolean
   - showInterstitial(): 播放插屏

3. 广告频次控制：
   - 每日激励广告上限：6次
   - 每日首次失败免费复活：1次
   - 插屏：每3关最多1次

4. 插屏跳过：
   - 第1章不显示插屏

完成后报告广告管理器结构。
```

### Day 18-2：5个广告场景接入

**Claude Code 指令**：
```
在对应位置接入广告：

1. 资源双倍（结算界面）：
   - 在 ResultPanel 中，点击"双倍奖励"按钮
   - 调用 AdManager.show(AD_UNITS.REWARD_DOUBLE, (completed) => {
       if (completed) { 奖励×2; }
     });

2. 原地复活（失败界面）：
   - 在 FailPanel 中，点击"复活"按钮
   - 调用 AdManager.show(AD_UNITS.RESURRECTION, (completed) => {
       if (completed) { 复活; 重置关卡状态; }
     });
   - 每日首次免费（AdManager 控制）

3. 稀有投掷物（高评价通关时）：
   - 在 ResultPanel 中，满足高评价（3星）时显示
   - 调用 AdManager.show(AD_UNITS.RARE_PROJECTILE, (completed) => {
       if (completed) { 给予稀有投掷物; }
     });

4. 每日礼包（首次打开游戏）：
   - 在 MainScene onLoad 中：
     检查是否今日首次打开
     调用 AdManager.show(AD_UNITS.DAILY_GIFT, (completed) => {
       if (completed) { 给予每日礼包金币; }
     });

5. 分享获资源（分享成功后）：
   - 在 ShareManager.ts 中，分享成功后：
     调用 AdManager.show(AD_UNITS.SHARE_REWARD, (completed) => {
       if (completed) { 给予50金币; }
     });

完成后报告5个广告场景接入情况。
```

### Day 19-2：填充率确认

**我执行**：
- 穿山甲广告填充率取决于：
  - 账号资质（个人账号 vs 企业账号）

**说明**：如果个人账号广告填充率低，可以后续考虑：
1. 接入 AdMob 作为备选（海外用户填充更好）

3. 降低对广告收入的预期，以用户增长为主要目标
