# 2026-03-26 日志

## 游戏项目：不破不立

### 项目概述
- **游戏名**：不破不立
- **类型**：物理建造+投掷破坏融合休闲游戏（Building + Throwing + Destruction）
- **引擎**：Cocos Creator 3.8 + TypeScript + Chipmunk物理引擎
- **目标**：微信小游戏，纯IAA（无内购）变现
- **核心玩法**：在有限材料预算下搭建结构 → 投掷物体击毁目标 → 体验破坏爽感

### 美术风格
- 温暖工业风：暖黄 #F5A623 + 深灰 #4A4A4A + 橙色 #FF6B35
- 美术方案：通义万相API生成（主力）+ Kenney.nl素材（补充）

### 章节规划
- 第1章"物理启蒙"（10关）：投掷基础、抛物线
- 第2章"结构秘密"（10关）：搭桥原理、支撑点
- 第3章"破坏艺术"（10关）：精准打击、连锁破坏
