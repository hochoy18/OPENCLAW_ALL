### 3. Google NotebookLM 详解
- 定位：AI 研究助手，只基于用户上传的文档回答问题，有据可查
- 核心功能：
  - 多源上传（PDF、Google Docs、网页、YouTube 视频转录、TXT/Markdown）
  - 问答摘要，自动标注来源
  - **Audio Overviews**（AI 播客对话）⭐ 最有特色的功能
  - 自动生成学习指南、FAQ、时间线等卡片
- 适合场景：学术研究、读书笔记、新闻追踪、事实核查、项目资料整理
- 局限：免费版用量限制、Audio Overview 目前主要是英文、无中文界面

### 4. NotebookLM + 视频制作
- NotebookLM 本身不做视频，是内容 + 配音工具
- 典型工作流：上传资料 → 生成 Audio Overview → 搭配虚拟人/剪辑工具做视频
- 工具组合：NotebookLM + HeyGen/Synthesia（虚拟人）+ 剪映/CapCut（剪辑）+ 翻译工具
- 适合：知识类短视频、读书/论文讲解、多语言翻译视频
