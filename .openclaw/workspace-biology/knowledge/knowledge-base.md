## 🧬 第1章 走近细胞 核心知识点

### 1.1 细胞是生命活动的基本单位

#### 细胞学说的建立

**建立者**：
- 施莱登(M.J. Schleiden, 1804-1881) - 德国植物学家
- 施旺(T. Schwann, 1810-1882) - 德国动物学家

**细胞学说要点**：
1. 细胞是一个有机体，一切动植物都由细胞发育而来，并由细胞和细胞产物所构成
2. 细胞是一个相对独立的单位，既有它自己的生命，又对与其他细胞共同组成的整体生命起作用
3. 新细胞是由老细胞分裂产生的

**细胞学说的意义**：
- 揭示了动物和植物的统一性，阐明了生物界的统一性
- 使人们认识到植物和动物有着共同的结构基础
- 在思想观念上打破植物学和动物学之间的壁垒
- 促使解剖学、生理学、胚胎学等学科获得共同基础
- 为后来进入分子水平打下基础
- 为生物进化论的确立埋下伏笔

#### 细胞学说的建立过程（科学史）

| 时间 | 科学家 | 贡献 |
|------|--------|------|
| 1543年 | 维萨里(A. Vesalius) | 解剖学研究，揭示人体在器官水平的结构 |
| 1665年 | 胡克(R. Hooke) | 用显微镜观察木栓组织，发现并命名"cell"细胞 |
| 1670s | 列文虎克(A. van Leeuwenhoek) | 自制显微镜，观察细菌、红细胞、精子等 |
| 1680s | 马尔比基(M. Malpighi) | 观察动植物微细结构（细胞壁、细胞质） |
| 1838年 | 施莱登 | 提出植物细胞学说 |
| 1839年 | 施旺 | 《关于动植物的结构及生长一致性的显微研究》 |
| 1858年 | 魏尔肖(R.L.C. Virchow) | 提出"所有细胞都来源于先前存在的细胞" |

**重要名言**：
- 施旺："现在，我们已推倒了分隔动植物界的巨大屏障。"
- 魏尔肖："所有的细胞都来源于先前存在的细胞。"
- 威尔逊(E.B. Wilson)："每一个生物科学问题的答案都必须在细胞中寻找。"

#### 生命系统的结构层次

**概念层级**（从微观到宏观）：
- **细胞** → **组织** → **器官** → **系统** → **个体**
- **种群** → **群落** → **生态系统** → **生物圈**

**生命系统的特征**：
- 系统：彼此间相互作用、相互依赖的组分有规律地结合而形成的整体
- 层次性：各级系统层层相依，又各自有特定的组成、结构和功能
- 基础性：细胞是基本的生命系统，各层次生命系统的形成、维持和运转都以细胞为基础

#### 归纳法
- **完全归纳法**：观察所有类型，得出结论
- **不完全归纳法**：观察部分类型，得出结论（可能存在例外）

### 1.2 细胞的多样性和统一性

#### 原核细胞与真核细胞

| 特征 | 原核细胞 | 真核细胞 |
|------|----------|----------|
| **细胞核** | 无成形的细胞核（无核膜） | 有成形的细胞核（有核膜） |
| **DNA** | 环状DNA分子，位于拟核区 | DNA与蛋白质结合形成染色体 |
| **细胞器** | 只有核糖体 | 多种细胞器 |
| **代表生物** | 细菌、蓝细菌 | 植物、动物、真菌 |

**常见原核生物**：
- 细菌（如大肠杆菌）
- 蓝细菌（旧称蓝藻）：色球蓝细菌、颤蓝细菌、念珠蓝细菌、发菜

**蓝细菌特点**：
- 含有藻蓝素和叶绿素，能进行光合作用
- 自养生物
- 直径约10μm，有的可达70μm
- 淡水富营养化可导致"水华"

**细胞大小**：
- 大多数细菌直径：0.5~5.0μm
- 蓝细菌直径：约10μm
- 人体细胞：多数10-100μm

#### 细胞的统一性
共同结构：细胞膜、细胞质、遗传物质（DNA）、核糖体

#### 高倍显微镜观察实验
**观察材料**：酵母菌、水绵、叶保卫细胞、鱼的红细胞等

**高倍镜使用步骤**：
1. 转动反光镜使视野明亮
2. 在低倍镜下观察清楚后，把要放大观察的目标移至视野中央
3. 转动转换器，换成高倍物镜
4. 用细准焦螺旋调焦并观察

## 📝 第2章 组成细胞的分子 核心知识点

### 2.1 细胞中的元素和化合物
- 组成细胞的元素都能在无机自然界中找到
- 大量元素：C、H、O、N、P、S、K、Ca、Mg等
- 微量元素：Fe、Mn、Zn、Cu、B、Mo等
- 基本元素：C
- 最基本元素：C、H、O、N

### 2.2 细胞中的无机物

#### 水
- 含量：占细胞重量的60%-90%
- 结合水：与细胞内其他物质结合
- 自由水：游离形式，可自由流动
- 功能：运输营养物质和废物、反应物、良好溶剂

#### 无机盐
- 存在形式：离子
- 功能：维持细胞和生物体的生命活动

### 2.3 细胞中的糖类和脂质

#### 糖类
| 类型 | 举例 | 功能 |
|------|------|------|
| 单糖 | 葡萄糖、果糖、半乳糖 | 直接能源 |
| 二糖 | 麦芽糖、蔗糖、乳糖 | 提供能量 |
| 多糖 | 淀粉、纤维素、糖原 | 储存/结构支持 |

#### 脂质
| 类型 | 功能 |
|------|------|
| 脂肪 | 储能、保温、缓冲 |
| 磷脂 | 构成细胞膜和细胞器膜 |
| 固醇 | 胆固醇、性激素、维生素D |

### 2.4 蛋白质是生命活动的主要承担者

**组成元素**：C、H、O、N，有的含P、S

**基本单位**：氨基酸（约20种）

**结构层次**：氨基酸 → 二肽 → 多肽 → 蛋白质

**蛋白质结构多样性的原因**：
- 氨基酸种类不同
- 氨基酸数目不同
- 氨基酸排列顺序不同
- 肽链折叠方式不同

**蛋白质的功能**：
- 结构蛋白：羽毛、头发、肌肉
- 功能蛋白：酶（催化）、抗体（免疫）、血红蛋白（运输）、激素（调节）

**相关科学史**：
- 1965年，中国合成结晶牛胰岛素 - 世界上第一个人工合成蛋白质

### 2.5 核酸是遗传信息的携带者

**核酸种类**：DNA（脱氧核糖核酸）、RNA（核糖核酸）

**功能**：储存、传递和表达遗传信息

**分布**：
- DNA：主要在细胞核
- RNA：主要在细胞质

**组成**：核苷酸（含氮碱基、磷酸、五碳糖）

## 🔬 第3章 细胞的基本结构 核心知识点

### 3.1 细胞膜的结构和功能

#### 细胞膜的流动镶嵌模型
- 磷脂双分子层：构成膜的基本支架
- 蛋白质：覆盖、嵌入、贯穿
- 糖蛋白：识别作用（糖被）

#### 细胞膜的功能特性
- 选择透过性

#### 细胞膜的功能
- 将细胞与外界环境分隔开
- 控制物质进出细胞
- 进行细胞间的信息交流

### 3.2 细胞器之间的分工合作

#### 双层膜细胞器
- **线粒体**：有氧呼吸的主要场所（嵴、基质）
- **叶绿体**：光合作用的场所（类囊体、基粒）

#### 单层膜细胞器
- **内质网**：蛋白质合成加工、脂质合成
- **高尔基体**：蛋白质加工、包装、分泌
- **液泡**：调节细胞内环境
- **溶酶体**：分解衰老损伤的细胞器

#### 无膜细胞器
- **核糖体**：蛋白质合成场所
- **中心体**：与有丝分裂有关

#### 细胞器之间的协调配合
- 分泌蛋白的合成和分泌

#### 用高倍显微镜观察叶绿体和细胞质的流动

### 3.3 细胞核的结构和功能

**结构**：
- 核膜：双层膜
- 核孔：实现核质物质交换
- 核仁：与核糖体RNA合成有关
- 染色质：DNA+蛋白质

**染色质与染色体**：同一物质在不同时期的两种形态

**功能**：细胞核是遗传信息库，是细胞代谢和遗传的控制中心

**生物科技进展**：世界上首例体细胞克隆猴的诞生（2017年11月27日）

## 💧 第4章 细胞的物质输入和输出 核心知识点

### 4.1 被动运输

#### 自由扩散
- 特点：不需要载体、不消耗能量
- 实例：O₂、CO₂、乙醇、甘油

#### 协助扩散
- 特点：需要载体、不消耗能量
- 实例：葡萄糖进入红细胞

#### 渗透作用
- 半透膜的特性
- 渗透的方向
- 动物细胞的吸水和失水
- 植物细胞的质壁分离

**探究植物细胞的吸水和失水**

**生物科学史话**：人类对通道蛋白的探索历程

### 4.2 主动运输与胞吞、胞吐

#### 主动运输
- 特点：需要载体、消耗能量
- 意义：逆浓度梯度运输物质
- 实例：离子、小肠吸收葡萄糖、氨基酸

#### 胞吞和胞吐
- 特点：大分子物质进出细胞的方式
- 胞吞：物质附着在细胞膜上 → 膜凹陷 → 形成囊泡
- 胞吐：分泌蛋白等物质从细胞膜排出

## ⚡ 第5章 细胞的能量供应和利用 核心知识点

### 5.1 降低化学反应活化能的酶

#### 一、酶的作用和本质
- 酶的概念
- 酶的本质：蛋白质或RNA
- 酶的作用机制：降低活化能

**实验**：比较过氧化氢在不同条件下的分解

#### 二、酶的特性
- 高效性：催化效率是无机催化剂的10⁷~10¹³倍
- 专一性：一种酶只能催化一种或一类反应
- 作用条件温和

**实验**：淀粉酶对淀粉和蔗糖的水解作用、影响酶活性的条件

### 5.2 细胞的能量"货币"ATP

#### ATP的结构
- 全称：腺嘌呤核糖核苷三磷酸
- 简式：A-P~P~P
- A：腺苷（腺嘌呤+核糖）
- P：磷酸基团
- ~：高能磷酸键

#### ATP与ADP的相互转化
- ATP → ADP + Pi + 能量（放能反应）
- ADP + Pi + 能量 → ATP（吸能反应）
- 特点：可逆反应，但能量不可逆

#### ATP的利用
- 用于主动运输
- 用于生物发电、发光
- 合成蛋白质、脂肪等有机物

### 5.3 细胞呼吸的原理和应用

#### 细胞呼吸的类型
- 有氧呼吸
- 无氧呼吸

#### 有氧呼吸
- 场所：细胞质基质 + 线粒体
- 总反应式：有机物 + O₂ → CO₂ + H₂O + 能量
- 三个阶段：
  1. C₆H₁₂O₆ → 丙酮酸 + [H]（细胞质基质）
  2. 丙酮酸 + H₂O → CO₂ + [H]（线粒体基质）
  3. [H] + O₂ → H₂O（线粒体内膜）

#### 无氧呼吸
- 场所：细胞质基质
- 总反应式：
  - 酒精发酵：C₆H₁₂O₆ → 酒精 + CO₂ + 少量能量
  - 乳酸发酵：C₆H₁₂O₆ → 乳酸 + 少量能量
- 实例：酿酒（酵母菌）、酸奶（乳酸菌）

**实验**：探究酵母菌细胞呼吸的方式

### 5.4 光合作用与能量转化

#### 一、捕获光能的色素和结构
- 叶绿体中的色素：
  - 叶绿素a（蓝绿色）
  - 叶绿素b（黄绿色）
  - 胡萝卜素（橙黄色）
  - 叶黄素（黄色）
- 色素的吸收光谱：主要吸收红光和蓝紫光

**实验**：叶绿体中色素的提取和分离

#### 二、光合作用的原理和应用
- 总反应式：CO₂ + H₂O → (CH₂O) + O₂
- 光反应：水的光解、ATP合成（NADP⁺ + H⁺ → NADPH）
- 暗反应（碳反应）：CO₂的固定、C₃的还原
- 环境因素对光合作用强度的影响

## 🔄 第6章 细胞的生命历程 核心知识点

### 6.1 细胞的增殖

#### 细胞周期
- 概念：连续分裂的细胞，从一次分裂完成时开始，到下一次分裂完成时为止
- 分裂间期：DNA复制、蛋白质合成
- 分裂期：包括前期、中期、后期、末期

#### 有丝分裂各时期特点
| 时期 | 特点 |
|------|------|
| 前期 | 核膜解体、核仁消失；出现染色体和纺锤体 |
| 中期 | 染色体形态稳定、数目清晰，着丝点排列在赤道板上 |
| 后期 | 着丝点分裂，姐妹染色单体分开，染色体数目加倍 |
| 末期 | 核膜重建、核仁出现；细胞质分裂，植物细胞形成细胞板 |

#### 植物细胞有丝分裂过程
- 与动物细胞的区别：前期（中心体）；末期（细胞板而非缢裂）

**实验**：观察根尖分生区组织细胞的有丝分裂

### 6.2 细胞的分化

#### 细胞分化的概念和特点
- 概念：在个体发育中，由一个或一种细胞增殖产生的后代，在形态、结构和生理功能上发生稳定性差异的过程
- 特点：持久性、不可逆性、全能性

#### 细胞的全能性
- 概念：已经分化的细胞，仍然具有发育成完整个体的潜能
- 实例：植物组织培养、动物克隆

#### 干细胞
- 概念：具有分裂和分化能力的细胞
- 实例：骨髓造血干细胞

### 6.3 细胞的衰老和死亡

#### 细胞衰老的特征
- 细胞膜通透性改变
- 细胞核体积增大
- 线粒体数量减少
- 色素积累

#### 细胞凋亡
- 概念：由基因决定的细胞自动结束生命的过程
- 实例：人胚胎发育时手指的形成、蝌蚪尾巴的消失
- 意义：对于生物体的正常发育和维持内部环境稳定具有重要作用

#### 细胞坏死
- 特点：不利因素影响下的细胞损伤和死亡
