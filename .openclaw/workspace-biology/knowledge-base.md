## 第1章 走近细胞

### 1.1 细胞是生命活动的基本单位

#### 细胞学说建立过程
- **建立者**：施莱登(1838)、施旺(1839) - 德国科学家
- **内容要点**：
  1. 细胞是一个有机体，一切动植物都由细胞发育而来，并由细胞和细胞产物所构成
  2. 细胞是一个相对独立的单位，既有它自己的生命，又对与其他细胞共同组成的整体生命起作用
  3. 新细胞是由老细胞分裂产生的
- **意义**：揭示了动物和植物的统一性，阐明了生物界的统一性

#### 施莱登和施旺的研究对象
- 施莱登：观察花粉、胚珠、柱头，发现都有细胞核
- 施旺：动物细胞

#### 魏尔肖的贡献
- 名言："所有的细胞都来源于先前存在的细胞"
- 证明新细胞通过分裂产生

#### 生命系统的结构层次
- **从小到大**：细胞 → 组织 → 器官 → 系统 → 个体 → 种群 → 群落 → 生态系统 → 生物圈
- **细胞是基本的生命系统**

### 1.2 细胞的多样性和统一性

#### 显微镜使用
- 低倍镜 → 高倍镜观察步骤
- 材料：酵母菌、水绵、叶保卫细胞、鱼红细胞等

#### 真核细胞 vs 原核细胞
| 特征 | 真核细胞 | 原核细胞 |
|------|----------|----------|
| 细胞核 | 有核膜包被的细胞核 | 无成形细胞核，只有拟核 |
| DNA | 染色体 | 环状DNA |
| 代表生物 | 植物、动物、真菌 | 细菌、蓝细菌(蓝藻) |

#### 蓝细菌特点
- 含藻蓝素和叶绿素，能进行光合作用
- 直径约10µm，有的可达70µm
- 水华：淡水污染富营养化导致大量繁殖

## 第2章 组成细胞的分子

### 2.1 细胞中的元素和化合物
- 组成细胞的元素都能在无机自然界中找到
- 大量元素：C、H、O、N、P、S、K、Ca、Mg等
- 微量元素：Fe、Mn、Zn、Cu、B、Mo等
- 基本元素：C
- 最基本元素：C、H、O、N

### 2.2 细胞中的无机物
- **水**：占细胞重量的60%-90%
  - 结合水：与细胞内其他物质结合
  - 自由水：游离形式，可自由流动
- **无机盐**：维持细胞和生物体的生命活动

### 2.3 细胞中的糖类和脂质
#### 糖类
- 单糖：葡萄糖、果糖、半乳糖
- 二糖：麦芽糖、蔗糖、乳糖
- 多糖：淀粉、纤维素、糖原
- 功能：主要能源物质

#### 脂质
- 脂肪：储能、保温、缓冲
- 磷脂：构成细胞膜和细胞器膜
- 固醇：胆固醇、性激素、维生素D

### 2.4 蛋白质是生命活动的主要承担者
- **组成元素**：C、H、O、N，有的含P、S
- **基本单位**：氨基酸（约20种）
- **结构**：氨基酸→多肽→蛋白质
- **功能**：
  - 结构蛋白：羽毛、头发
  - 功能蛋白：酶（催化）、抗体（免疫）、血红蛋白（运输）、激素（调节）
- **相关科学史**：世界上第一个人工合成蛋白质（1965年，中国合成结晶牛胰岛素）

### 2.5 核酸是遗传信息的携带者
- **核酸种类**：DNA（脱氧核糖核酸）、RNA（核糖核酸）
- **功能**：储存、传递和表达遗传信息
- **分布**：DNA主要在细胞核，RNA主要在细胞质
- **组成**：核苷酸（含氮碱基、磷酸、五碳糖）

## 第3章 细胞的基本结构

### 3.1 细胞膜的结构和功能
- **结构**：磷脂双分子层（基本支架）+ 蛋白质 + 糖被（糖链）
- **功能**：
  - 将细胞与外界环境分隔开
  - 控制物质进出（选择透过性）
  - 进行细胞间信息交流
- **结构特点**：流动性（磷脂分子和蛋白质分子可以运动）

### 3.2 细胞器之间的分工合作
#### 八大细胞器
1. **线粒体**：有氧呼吸主要场所，"动力车间"
2. **叶绿体**：光合作用场所，"养料制造车间"和"能量转换站"
3. **内质网**：蛋白质合成和加工，脂质合成
4. **高尔基体**：蛋白质加工、分类、包装、转运
5. **核糖体**：蛋白质合成场所，"生产蛋白质的机器"
6. **溶酶体**：分解衰老、损伤的细胞器，"消化车间"
7. **液泡**：调节细胞内环境
8. **中心体**：与细胞分裂有关

#### 生物膜系统
- 内质网、高尔基体、线粒体、叶绿体、溶酶体等通过膜连接
- 细胞器膜和细胞膜、核膜等共同构成细胞的生物膜系统

### 3.3 细胞核的结构和功能
- **结构**：核膜（双层膜）、核孔、核仁、染色质
- **功能**：遗传信息库，是细胞代谢和遗传的控制中心
- **DNA**：主要的遗传物质
- **染色质**：DNA+蛋白质
- **相关科学史**：克隆猴（中科院，2017年）

## 第4章 细胞的物质输入和输出

### 4.1 被动运输
- **自由扩散**：顺浓度梯度，不需要载体和能量（如O₂、CO₂、甘油、乙醇）
- **协助扩散**：顺浓度梯度，需要载体蛋白（如葡萄糖进入红细胞）
- **渗透作用**：水分子通过半透膜的扩散

#### 探究实验：植物细胞的吸水和失水
- 质壁分离现象
- 条件：原生质层（细胞膜+液泡膜+两者之间的细胞质）比细胞壁伸缩性大

### 4.2 主动运输与胞吞、胞吐
- **主动运输**：逆浓度梯度，需要载体和能量
  - 例：离子、小分子主动运输
- **胞吞、胞吐**：大分子物质进出细胞
  - 需要能量
  - 例：蛋白质、细菌等

## 第5章 细胞的能量供应和利用

### 5.1 降低化学反应活化能的酶
- **酶的本质**：活细胞产生的具有催化作用的有机物（大多数是蛋白质，少数是RNA）
- **作用机理**：降低化学反应所需的活化能
- **特点**：高效性、专一性、作用条件温和
- **相关实验**：探究影响酶活性的条件（温度、pH）
- **科学史**：人类对通道蛋白的探索历程

### 5.2 细胞的能量"货币"ATP
- **ATP结构**：腺苷三磷酸（A-P~P~P）
- **特点**：不稳定，含有两个高能磷酸键
- **功能**：细胞生命活动的直接能源物质
- **合成途径**：呼吸作用（分解有机物）和光合作用（光反应）

### 5.3 细胞呼吸的原理和应用
- **有氧呼吸**：
  - 场所：细胞质基质 + 线粒体
  - 总反应式：C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量
  - 三阶段：
    1. 葡萄糖 → 丙酮酸 + [H]（细胞质基质）
    2. 丙酮酸 + H₂O → CO₂ + [H]（线粒体基质）
    3. [H] + O₂ → H₂O（线粒体内膜）
- **无氧呼吸**：
  - 场所：细胞质基质
  - 产物：酒精+CO₂ 或 乳酸
- **探究实验**：探究酵母菌细胞呼吸的方式

### 5.4 光合作用与能量转化
#### 光合色素的提取和分离
- **实验**：绿叶中色素的提取和分离
- **色素种类**：
  - 叶绿素a（蓝绿色）：主要吸收红光和蓝紫光
  - 叶绿素b（黄绿色）：主要吸收蓝紫光
  - 胡萝卜素（橙黄色）
  - 叶黄素（黄色）
- **分离方法**：纸层析法

#### 光合作用过程
- **光反应**：场所（类囊体薄膜）
  - 物质变化：H₂O → O₂；ADP + Pi → ATP；NADP⁺ + H⁺ → NADPH
  - 能量变化：光能 → ATP中活跃化学能
- **暗反应**（碳反应）：场所（叶绿体基质）
  - 物质变化：CO₂ → C₃ → (CH₂O)等有机物
  - 能量变化：ATP中活跃化学能 → 有机物中稳定化学能
- **总反应式**：6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂

#### 影响因素
- 光照强度、温度、CO₂浓度、叶龄等
- **探究实验**：探究环境因素对光合作用强度的影响

## 第6章 细胞的生命历程

### 6.1 细胞的增殖
- **细胞周期**：分裂间期（G₁、S、G₂）+ 分裂期（M）
- **有丝分裂**：
  - 染色体复制一次，细胞分裂一次
  - 过程：前期（核膜消失、出现染色体）→ 中期（染色体排列在赤道板上）→ 后期（着丝粒分裂，姐妹染色单体分开）→ 末期（核膜重建，细胞质分裂）
  - 特点：染色体和DNA数目变化规律
- **探究实验**：观察根尖分生区组织细胞有丝分裂

### 6.2 细胞的分化
- **概念**：在个体发育中，由一个或一种细胞增殖产生的后代，在形态、结构和生理功能上发生稳定性差异的过程
- **特点**：
  - 普遍性
  - 持久性
  - 稳定性
  - 不可逆性
- **本质**：基因选择性表达（管家基因 + 组织特异性基因）
- **意义**：是生物个体发育的基础

### 6.3 细胞的衰老和凋亡
#### 细胞衰老
- 特征：
  - 细胞内水分减少，萎缩
  - 酶活性降低（头发变白）
  - 色素积累
  - 细胞膜通透性改变
  - 呼吸速率减慢
- 与个体衰老的关系：单细胞生物→细胞衰老=个体衰老；多细胞生物→细胞衰老≠个体衰老

#### 细胞凋亡（编程性死亡）
- 概念：由基因决定的细胞自动结束生命的过程
- 实例：人胚胎发育过程中尾的消失、蝌蚪尾的消失
- 与细胞坏死的区别

### 6.4 细胞的癌变
- **概念**：有的细胞受到致癌因子的作用，细胞中遗传物质发生变化，变成不受机体控制的、连续分裂的恶性增殖细胞
- **致癌因子**：
  - 物理因子：X射线、紫外线等
  - 化学因子：亚硝胺、黄曲霉毒素等
  - 病毒因子：HPV、HBV等
- **特征**：
  - 无限增殖
  - 形态结构改变
  - 表面糖蛋白减少，黏着性降低
- **预防**：健康生活方式
- **相关科学史**：世界上首例体细胞克隆猴
